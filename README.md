# 🍦 Ice Cream App

## 📌 Overview
Ice Cream App is a modern web application where users can browse different ice cream flavors, view product details, and use cart functionality.

This project uses **Sanity CMS** for dynamic content management, allowing products to be easily managed from the backend without changing the frontend code.

---

## 🚀 Features
- Home page with ice cream product listing
- Dynamic product data from Sanity CMS
- Category/flavor-based browsing
- Product detail page
- Add to cart functionality (if implemented)
- Responsive UI design
- Smooth navigation between pages

---

## 🛠️ Tech Stack

### Frontend
- React.js / Next.js 
- HTML, CSS, JavaScript
- Tailwind CSS / Bootstrap / Custom CSS

### Backend / CMS
- **Sanity CMS**
  - Product data management
  - Real-time content updates
  - Structured schemas for ice cream products

### Other Tools
- React Router (if used)
- Context API / Redux (if used)
- Git & GitHub

---


## 🧠 Sanity CMS Usage

- Products are stored in Sanity backend  
- Frontend fetches data using GROQ queries  
- Easy content updates without redeploying the app  

### 📦 Schemas used:
- Ice cream products  
- Categories  
- Images & pricing
